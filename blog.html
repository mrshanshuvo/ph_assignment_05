Question-1: What are the different ways to select an element in the DOM?

getElementById("id") - Selects one element by ID.
getElementsByClassName("class") - Selects multiple elements by class.
getElementsByTagName("tag") - Selects multiple elements by tag (like p, div).
querySelector("selector") - Selects the first matching element using a CSS selector.
querySelectorAll("selector") - Selects all matching elements using a CSS selector.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML - Gets or sets HTML inside an element (includes tags).
innerText - Gets or sets only the visible text (ignores hidden text).
textContent - Gets or sets all text (even hidden ones).

Question-3: What is event delegation in the DOM?

Instead of adding an event to many elements, we add it to their parent. The parent listens for clicks and handles them.

Question-4: What is event bubbling in the DOM?

When you click an element, the event first happens on it, then moves up to its parent and so on.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element in JavaScript, use document.createElement("tag"). To add it to the page, use appendChild(). To remove an element, use remove().
Example:

let newElement = document.createElement("p");
newElement.textContent = "Hello!";
document.body.appendChild(newElement);
newElement.remove();
